# Jedi vs Sith Battle Simulator
A CLI based battle simulator with CSV database.
## Features
- Choose two fighters to battle
- Choose one fighter to fight all 4 opponents in Gauntlet mode
- Randomly generated fights have unpredictable outcomes.
- Fighter have chance to use special abiltiy, Rage or Deflect
- Cinematic announcing throughout
- Ability to view individual Fighters stats
- Keeps record of Wins, Losses and Gauntlets wins (Able to reset)
- Imports 2 CSV files with Jedi and Sith stats.
